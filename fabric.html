Instalace Fabric

Instalace

  • 1. Nastavit Linux (nejlépe Debiana/Ubuntu)
  • 2. sudo apt update
  • 3. sudo apt upgrade
  • 4. git clone https://github.com/danielmiessler/fabric.git
  • 5. cd fabric
  • 6. sudo apt install pipx
  • 7. pipx install .
  • 8. pipx ensurepath
  • 9. source ~/.bashrc
  • 10. fabric --setup
  • 10.1 Přidat YouTube API klíč

Použití

  • yt --transcript odkaz_na_yt_video | fabric --model llama3:latest -sp extract_wisdom
