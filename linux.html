Zde jsou dobré a užitečné příkazy pro Linux:

Tyto příkazy nemusí fungovat na všech distribucích Linuxu!


Systémové příkazy

  • cal
  • neofetch

Správa balíčků (Pacman)

  • sudo pacman -Syu
  • sudo pacman -S názevaplikace
  • sudo pacman -S bluez blueman bluez-utils p7zip unrar tar rsync git neofetch exfat-utils fuse-exfat ntfs-3g flac aria2 curl wget git base-devel

Správa balíčků (Yay)

  • git clone https://aur.archlinux.org/yay.git
  • cd yay
  • makepkg -si
  • yay --version
  • yay -S názevaplikace
  • yay -Rs názevaplikace

Správa balíčků (APT)

  • sudo apt install cesta k souboru nebo aplikace z repozitáře
  • sudo apt install git
  • sudo apt install build-essential<\li>
  • sudo apt install hashcat
  • sudo apt install cupp
  • sudo apt install samba

Práce se soubory a adresáři

  • sudo rm
  • sudo rm -rf
  • sudo rmdir
  • sudo mkdir
  • touch nazevsouboru.pripona
  • sudo nano nazevsouboru

Správa uživatelů a oprávnění

  • sudo chmod -R 777
  • [nazevsdileneslozky]
    path = /cesta/k/sdilene/slozce
    writable = yes
    guest ok = yes
  • sudo smbpasswd -a uzivatelske_jmeno

Speciální příkazy

  • cupp -i
